# Requisitos Funcionais

Indicação de níveis de importância (Exemplo: P0 = Indispensável, P1 = Deveria ter, e P2 = Seria bom ter) e a descrição do que a funcionalidade deveria fazer.

| Prioridade | Descrição                                                                                                                                                                                                                                                 |
| :--------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| P0         | Autenticação e gerenciamento dos usuários, o sistema deverá permitir que o usuário crie uma conta, faça login e logout e mantenha os dados associados ao seu perfil.                                                                                      |
| P0         | Criação de perfil do usuário, o sistema deverá permitir que o usuário insira um username único, foto de perfil opcional e data de nascimento.                                                                                                             |
| P0         | Persistência de dados, o sistema deverá persistir os dados relacionados ao usuário como o cronograma, resultados de simulados, histórico de resolução de questões e preferências da conta como temas visuais.                                             |
| P0         | Modo simulado onde o aluno poderá simular uma prova do ENEM com cronômetro, escolha do idioma, escolha das áreas de conhecimento. Com feedback no final do simulado.                                                                                      |
| P1         | Análise de questões pós-simulado com uma nota estimada, tempo médio gasto por questão, questões corretas, erradas e não respondidas, desempenho por área e uma lista de erros com o enunciado da questão, alternativa correta e a escolhida pelo usuário. |
| P0         | Modo de estudo onde o aluno pode escolher as questões que deseja responder (por ano, área de conhecimento, idioma) e tenha um feedback visual ao responder cada questão.                                                                                  |
| P2         | Histórico de resolução de questões para gerar estatísticas simples sobre onde o aluno precisa melhorar, porcentagem de acertos por área de conhecimento, idiomas e anos.                                                                                  |
| P0         | Cronograma semanal onde o estudante pode distribuir as áreas do conhecimento durante os dias da semana para se organizar melhor.                                                                                                                          |
| P0         | Distribuição de questões diárias por área baseado no cronograma do estudante.                                                                                                                                                                             |
| P0         | Configurações do usuário onde ele poderá escolher o idioma preferido (inglês ou espanhol), número de questões que deseja praticar por dia (por área).                                                                                                     |
| P2         | Marcar questões para responder depois e salvá-las em uma lista de questões salvas pelo usuário, com filtros de questões erradas, certas e não respondidas.                                                                                                |
| P2         | Meta do curso desejado onde o estudante poderá escolher uma Universidade e curso desejado e a última nota de corte da modalidade em que está inserido.                                                                                                    |
| P2         | Preferências de temas onde o usuário poderá escolher entre Light, Dark e Default (OS).                                                                                                                                                                    |
